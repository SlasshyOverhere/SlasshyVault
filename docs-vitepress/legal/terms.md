## 2. Description

SlasshyVault is a desktop application that indexes media files from your local storage and Google Drive, and provides an interface to browse, search, and play them. The Software is provided free of charge for personal, non-commercial use under the MIT license.

## 3. Google Drive Integration

The Software uses Google OAuth to authenticate with your Google Drive account solely to list and stream media files you explicitly choose to index. It does not upload, modify, or delete files without your explicit action. OAuth tokens are stored locally and never sent to any server other than Google.

## 4. Third-Party Services

The Software may call TMDB, imdbapi.dev, or OMDb to enrich metadata. These calls use your own API keys or free public endpoints. No personal information is sent. The Watch Together relay runs on your own Cloudflare account.

## 5. No Data Collection

SlasshyVault does not collect telemetry, analytics, usage statistics, or personal information. There are no servers operated by the developers that receive data from the Software. The OAuth backend retains no user data beyond 5 minutes.
