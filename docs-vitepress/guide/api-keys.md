# API Keys

| Service | Default | Custom Key? |
|---|---|---|
| TMDB (metadata) | — | ✅ Recommended |
| OMDb (IMDb ratings) | imdbapi.dev (free) | ✅ Optional |

Get a free TMDB key at [themoviedb.org/settings/api](https://www.themoviedb.org/settings/api). Enter it in Settings → API Keys.

IMDb ratings fall back to the free [imdbapi.dev](https://imdbapi.dev) service if no OMDb key is provided.
