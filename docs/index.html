1. Acceptance

By downloading, installing, or using SlasshyVault (the "Software"), you agree to these Terms. If you do not agree, do not use the Software.

2. Description

SlasshyVault is a desktop application that indexes media files from your local storage and Google Drive, and provides an interface to browse, search, and play them via external players such as MPV. The Software is provided "as is" for personal, non-commercial use.

3. Google Drive Integration

The Software uses Google OAuth to authenticate with your Google Drive account solely to list and stream media files you explicitly choose to index. The Software does not upload, modify, or delete any files in your Drive without your explicit action. OAuth tokens are stored locally on your device and are never sent to any server other than Google's authorized endpoints.

4. Third-Party Services

The Software may optionally call third-party APIs (TMDB, IMDb, OMDb) to enrich metadata. These calls use either your own API keys or free public endpoints. No personal information is sent to these services. The Software also supports deploying a Cloudflare Worker for the Watch Together feature; this Worker runs on your own Cloudflare account and never communicates with any central server.

5. No Data Collection

SlasshyVault does not collect telemetry, analytics, usage statistics, or personal information. There are no servers operated by the developers that receive any data from the Software. The OAuth backend server only handles Google's authentication redirect and retains no user data beyond the lifetime of a single OAuth session (5 minutes).

Privacy Policy

Last updated: June 30, 2026

What We Collect

Nothing. SlasshyVault does not collect, store, transmit, or share any personal data. There are no analytics, no telemetry, no crash reporters, no advertising identifiers, no fingerprinting, and no user tracking of any kind.

Google Drive Access

When you sign in with Google, the Software requests access to your Google Drive to list and stream media files. This access is used exclusively for:

  • Listing folders and files you choose to index
  • Streaming video content directly to your media player
  • Reading watch-history metadata previously saved to your Drive (if any)

The Software never writes, modifies, or deletes files in your Drive without your explicit action. Your OAuth tokens are stored encrypted on your local machine and are never sent to any third party.

OAuth Backend Server

The OAuth backend server (auth.slasshyvault.com or self-hosted) handles only the Google OAuth code exchange. It stores OAuth state and session IDs in memory for a maximum of 5 minutes, after which they are permanently deleted. No user information is logged or persisted.

Watch Together Relay

The Watch Together feature uses a Cloudflare Worker deployed to your own Cloudflare account. This Worker relays WebSocket messages between participants in a Watch Together room. It stores no data beyond the active room session (Durable Object in-memory state). You can delete the Worker at any time from your Cloudflare dashboard or through the Software settings.

Third-Party API Calls

The Software may make requests to TMDB (api.themoviedb.org), OMDb (www.omdbapi.com), or IMDb (api.imdbapi.dev) to fetch metadata such as titles, posters, ratings, and cast information. These requests are made directly from your device and include only the search query or media ID — no personal information is sent. These services have their own privacy policies.

Cloudflare Relay Deploy

If you use the one-click deploy feature, the Software makes API calls to api.cloudflare.com using the API token you provide. This deploys a Worker script to your Cloudflare account. The token is stored locally in your Software settings and is never transmitted to anyone other than Cloudflare.

Data Storage

  • Configuration: Stored locally in media_config.json in the app data directory.
  • Watch History: Stored in a local SQLite database on your device.
  • OAuth Tokens: Stored encrypted in local app data, used only for Google Drive API calls.
  • Cloudflare Token: Stored in local config if you choose to use the relay deploy feature.

Data Deletion

All locally stored data can be deleted from the Software settings (Factory Reset). OAuth tokens can be revoked at any time through your Google Account permissions page. The Cloudflare Worker can be deleted from the Software settings or directly from your Cloudflare dashboard.

Children

The Software is not intended for children under 13. No data is collected from any user regardless of age.
